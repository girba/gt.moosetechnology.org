Inspector

The GT Inspector lets you define custom presentations for each and every object. And then it enables you to navigate through them. Inexpensively.

Learn more:
humane-assessment.com/blog/the-moldable-gtinspector-deconstructed
scg.unibe.ch/research/moldableinspector

Spotter

The GT Spotter lets you search through objects. It is extensible like the inspector. Each object can define its own search processors.

Learn more:
humane-assessment.com/blog/introducing-gtspotter
humane-assessment.com/blog/boosting-gtspotter-with-preview

Playground

The GT Playground powers up the workspace with an inspector and makes the code part of the flow.

Learn more:
humane-assessment.com/blog/introducing-the-gtplayground
humane-assessment.com/blog/rethinking-print-it-in-pharo
humane-assessment.com/blog/rethinking-compilation-notifications-in-pharo

Debugger

The GT Debugger lets you build custom debuggers that can be switched to during debugging time.

Learn more:
scg.unibe.ch/research/moldabledebugger
humane-assessment.com/blog/debugging-tests-and-parsers-with-gtdebugger-demo

Download

Download a premade image on top of Pharo 5.0.

Or install it by yourself in a Pharo 4.0 image:

Gofer new
     smalltalkhubUser: 'Moose' project: 'GToolkit';
     configuration;
     loadStable.
#GTImageSetupCommandLineHandler asClass new activateWithoutSaving
